IMU Data Acquisition and Storage Experiment Based on Seeed XIAO and JY901

This is an experimental report summarizing the software and hardware design and results of the XIAO + JY901 system previously built.

Experiment Objectives

  1. Learn how to connect the Seeed XIAO development board with the JY901 IMU module.
  2. Implement real-time IMU data acquisition and waveform visualization.
  3. Store collected IMU data to an SD card in real time.
  4. Analyze the motion characteristics when the development board is shaken.

Equipment Used

  • Seeed XIAO development board ×1
  • Seeed XIAO expansion board (with SD card slot) ×1
  • JY901 IMU module ×1
  • Micro SD card ×1
  • Jumper wires × several
  • USB data cable ×1
  • Computer ×1

Hardware Connections

Connection Diagram

Pin Assignments

  • JY901 Communication: Software Serial D6(RX), D7(TX)
  • SD Card: SPI interface (integrated on the expansion board)

Experimental Procedure

  1. Hardware Preparation

    • Insert the SD card into the expansion board slot.
    • Connect the JY901 module according to the wiring diagram.
    • Connect the board to the computer via USB.

  2. Software Configuration

    • Install required Arduino libraries: SoftwareSerial, SPI, SD.
    • Upload the program to Seeed XIAO.
    • Open the Serial Monitor (115200 baud rate).

  3. Data Acquisition

    • The system automatically initializes the SD card and creates a data file.
    • Shake or rotate the board manually to collect motion data.
    • Observe real-time waveforms using the Serial Plotter.

  4. Data Management

    • Type “status” in the serial monitor to view the system status.
    • Type “stop” to stop recording.
    • Remove the SD card and read the stored data file.
